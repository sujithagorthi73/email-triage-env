\# Smart Email Triage Environment



In this project, I built a simple environment where an agent classifies emails into categories like work, spam, important, and personal.



\## Features

\- reset(), step(), state() functions

\- reward-based feedback system

\- easy, medium, hard difficulty levels

\- confidence score for predictions

\- explanation for each decision

\- priority tagging



\## How it works

The environment gives an email as input, and the agent selects a category.



Rewards:

\- Correct → +1

\- Close → +0.5

\- Wrong → -1



\## My Approach

I chose this problem because managing emails is something I deal with regularly.



I started simple, then improved it with reward logic and difficulty levels.



\## Future Improvements

\- Real email integration

\- ML-based classification



\## Conclusion

This project shows how agents can learn from structured environments.
